## Oracle server

### Setup: Serverless and AWS credentials

[https://www.serverless.com/framework/docs/providers/aws/guide/credentials/](https://www.serverless.com/framework/docs/providers/aws/guide/credentials)

### Run locally

```bash
cd oracle
npm i
npm start
```

## Front Web App

### Install

> Required: Node.js

```bash
cd front
npm install
```

### Run locally

```bash
cd front
npm start
```
